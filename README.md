## Install Laravel ver.7
`composer create-project --prefer-dist laravel/laravel:^7.0 slotegrator`

## Preparing for launch
- Set database connection settings in the .env file 
- $ `composer require laravel/ui:^2.4`
- $ `php artisan ui:auth`
- $ `php artisan migrate`

## Start Server
- $ `php artisan serve`
